## JSX란?

**JSX(JavaScript XML)**는 React의 개발 목적인 모든 HTML 코드를 하나의 JS파일에 넣어서 컴포넌트화시키는 것을 실현하기 위해 개발된 문법이다. \

JSX는 HTML 코드를 마치 XML을 코딩하는 것처럼 JS 파일에 넣을 수 있게 만들어 별도의 HTML 파일을 호출할 필요 없이 **컴포넌트의 독립성**을 유지시키기에 최적화된 문법이다.
